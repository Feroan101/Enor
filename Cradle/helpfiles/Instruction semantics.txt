PUSH - 0x01:
    consumes 1 operand byte
    effect: pushes x into stack
    ip += 2

POP - 0x02:
    consumes 1 operand byte 
    effect: pops x from the stack and stores x at 'pop_value'
    ip += 1 

ADD - 0x03:
    consumes 2 operand bytes
    requires: stack size >= 2 
    effect: pop 'val_1' pop 'val_2', push 'result' 
    ip += 1
